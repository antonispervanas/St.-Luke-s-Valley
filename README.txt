| CODE |
=======================================================================================================
Στον φάκελο Code θα βρείτε όλες τις κλάσεις του κώδικα με αρκετά απλές μεθόδους τύπου get/set.
Επιπλέον θα βρείτε αρχικοποιήσεις μεταβλητών, διάβασμα αρχείων και αποθήκευση πληροφοριών σε λίστες.
Ο κώδικας είναι εκτελέσιμος και δημιουργεί τις πρέπουσες λίστες αλλά δεν εμφανίζει κάτι.
=======================================================================================================


| EXECUTABLE |
=======================================================================================================
Στον φάκελο Executable στην παράδωση στο eclass θα βρείτε ένα εκτελέσιμο αρχείο "St. Luke's Valley.exe" που αποτελεί ένα πολύ
αρχικού επιπέδου Demo για το παιχνίδι και είναι δημιουργημένο σε Unity. Το Executable δεν έχει συνδεθεί
ακόμη άμεσα με τον κύριο κώδικα(CODE) και αυτός είναι και ο λόγος που ο φάκελος CODE περιέχει μόναχα
αρχικοποιήσεις και απλές μεθόδους. Επιλέξαμε να ασχολιθούμε περισσότερο με το Unity διότι δεν γνωρίζαμε
ακριβώς πως θα γίνει η σύνδεση και ποιες ακριβώς είναι οι δυνατότητες που μας παρέχει.
=======================================================================================================


| FILES |
=======================================================================================================
Στον φάκελο Files θα βρείτε τα ενδεικτικά αρχεία(.csv) που χρησιμοποιήσαμε και εμείς προκυμένου να
ελέγξουμε τον κώδικα στον φάκελο CODE. Μπορείτε αν θέλετε να τα κατεβάσετε και να αντικαταστήσετε τα
ανάλογα paths στην κλάση Program(main) εάν θέλετε να τον ελέγξετε και μόνοι σας.(πρέπει να
αντικαταστήσεται και τα paths μέσα στο αρχείο Characters για να αντιστοιχούν στον κατάλληλο διάλογο)
=======================================================================================================
